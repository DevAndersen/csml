# `<PrefixDecrement>`

## Description

Performs a decrement operation (prefix) on `Target`.

## Attributes

| Name | Type | Default | Mandatory | Description |
|---|---|---|---|---|
| `Target` | `string` | *None* | Yes | The target of the operation. |

## Elements

None.

## C# equivalent

The `--` operator (prefix).
