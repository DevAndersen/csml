# `<LessThan>`

## Description

Performs a less-than comparison on `<Left>` and `<Right>`.

## Attributes

None.

## Elements

- `Left`: The left-hand expression.
- `Right`: The right-hand expression.

## C# equivalent

The `<` operator.

## Example

### Câ™¯ML

```xml
<LessThan>
    <Left>
        <Value Value="2" />
    </Left>
    <Right>
        <Value Value="3" />
    </Right>
</LessThan>
```

### C#

```csharp
2 < 3
```
