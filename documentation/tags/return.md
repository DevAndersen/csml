# `<Return>`

## Description

Represents a `return` statement.

## Attributes

| Name | Type | Default | Mandatory | Description |
|---|---|---|---|---|
| `Value` | `string` | *Nothing* | No | The value to be returned. |

## Elements

None.

## C# equivalent

The `return` keyword.

## Example

```xml
<Return Value="123"/>
```

### C#

```csharp
return 123;
```
