# `<Remainder>`

## Description

Performs a remainder/modulo operation on `<Left>` and `<Right>`.

## Attributes

None.

## Elements

- `Left`: The left-hand expression.
- `Right`: The right-hand expression.

## C# equivalent

The `%` operator.

### Câ™¯ML

```xml
<Remainder>

        <Value Value="2" />
    </Left>
    <Right>
        <Value Value="1" />
    </Right>
</Remainder>
```

### C#

```csharp
2 - 1
```
