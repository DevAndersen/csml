# `<NotEquals>`

## Description

Performs an inequality comparison on `<Left>` and `<Right>`.

## Attributes

None.

## Elements

- `Left`: The left-hand expression.
- `Right`: The right-hand expression.

## C# equivalent

The `!=` operator.
