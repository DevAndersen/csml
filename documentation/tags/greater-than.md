# `<GreaterThan>`

## Description

Performs an greater-than comparison on `<Left>` and `<Right>`.

## Attributes

None.

## Elements

- `Left`: The left-hand expression.
- `Right`: The right-hand expression.

## C# equivalent

The `>` operator.

## Example

### Câ™¯ML

```xml
<GreaterThan>
    <Left>
        <Value Value="3" />
    </Left>
    <Right>
        <Value Value="2" />
    </Right>
</GreaterThan>
```

### C#

```csharp
3 > 2
```
