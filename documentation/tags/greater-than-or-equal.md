# `<GreaterThanOrEqual>`

## Description

Performs a greater-than-or-equal comparison on `<Left>` and `<Right>`.

## Attributes

None.

## Elements

- `Left`: The left-hand expression.
- `Right`: The right-hand expression.

## C# equivalent

The `>=` operator.

### Câ™¯ML

```xml
<GreaterThanOrEqual>

    </Left>
    <Right>
        <Value Value="2" />
    </Right>
</GreaterThanOrEqual>
```

### C#

```csharp
3 >= 2
```
